Intro to Web Apps

1. Make the
Hosts Talk

DNS

A Domain Name Server looks up the Internet address of the requested host domain, and directs the connection accordingly.

Ports

Computers have thousands of network Ports. These ports send and recieve messages over a network.

TCP/IP

Transmission Control Protocol builds upon basic Internet Protocol. These protocols allow port-to-port data transfer across a network.

2. Make the
Apps Talk

HTTP

Web Clients and Servers communicate using HyperText Transfer Protocol. This protocol manages the exchange of messages between the applications.

Request/Response

Using HTTP, a Client makes a request to a Server, which sends back a response.

3. Make the
Apps Interact

HTTP Methods

An HTTP request has a method, which expresses the action of the request.

GET

POST

Recap

Port: sends and recieves network data.

TCP/IP: transfers data across a network.

HTTP: passes messages between web applications.

GET: requests to access data.

POST: requests to submit data.
